This repo contains the code to plot the figures appearing in the paper Quantum many-body scars in random unitary circuits.
It takes the form of a single Jupyter notebook where each part of the code is explained with a Markdown cell.
The code is structured as follows:
1) Computation of the basic building blocks of the system necessary to evolve it.
2) Evolution of the system with codes that give a series of coefficients which determine the state of the system at time t.
3) Computation of purity and related figures.
4) Computation of OTOC and related figure.
